# Sakatl — app de ejercicio colaborativa

Next.js 16 (App Router) + TypeScript + Tailwind CSS v4.

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

- `/` — landing pública (implementada desde `Home.dc.html`, ver `figma.md`)
- `/app` — placeholder de la app de rutinas (pendiente de implementar desde `Fitness App.dc.html`)

## Datos de ejercicios

`public/exercises/` contiene el dataset completo de
[hasaneyldrm/exercises-dataset](https://github.com/hasaneyldrm/exercises-dataset)
(1,324 ejercicios, thumbnails, GIFs, instrucciones en 10 idiomas). Ver
`brief.md` para más contexto.

## Documentación del proyecto

- `brief.md` — descripción, funcionalidades, stack
- `figma.md` — referencias del diseño (Claude Design) y estado de cada pantalla
- `status.md` — estado y avance por agente
- `design-tokens/tokens.json` — tokens extraídos del diseño
